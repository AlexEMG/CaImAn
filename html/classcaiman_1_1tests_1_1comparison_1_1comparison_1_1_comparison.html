Detailed Description

Comparison It is used for comparison the modification on the CaImAn program.

You need to compare your modification to the groundtruth and then rename the folder created as "proof" for it to be send in you next push request

Here it has been made for 3 different functions. for it to compare well you need to set your ground truth with the same computer with which you are comparing the files class you instanciate to compare the different parts of CaImAn
